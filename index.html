CureHub

CureHub - Medical Store Management Application Overview: CureHub is a comprehensive medical store management application designed to streamline inventory management and facilitate efficient operations for medical store owners. Developed using Django as the backend framework, CureHub offers two versions: one with a RESTful API to manage the medical store's data and another without API for direct user interaction. With features such as user authentication, medicine management, and search functionality, CureHub empowers store owners to effectively manage their inventory and serve their customers efficiently.
